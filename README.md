# 🎮 Solo Leveling Finance: Gerenciamento de Gastos Gamificado

Uma aplicação web interativa que transforma a educação financeira e o controle de gastos em uma experiência de RPG. Desenvolvido com React, Tailwind CSS e TypeScript.

---

## Por que este projeto importa? (O Impacto na Comunidade)

A educação financeira no Brasil e no mundo enfrenta uma crise de engajamento. As planilhas tradicionais e os aplicativos bancários convencionais são frios, punitivos e projetados para um público que já possui disciplina financeira. 

Para a comunidade em geral, e **especialmente para as faixas etárias mais novas (15 a 27 anos)**, o modelo tradicional falha por um motivo simples: ele não fala a linguagem deles. É aqui que o **Solo Leveling Finance** entra para mudar o jogo.

### 1. Traduzindo o Abstrato para o Familiar 
Jovens cresceram imersos em videogames, onde conceitos complexos de gerenciamento de recursos são aprendidos intuitivamente através de mecânicas de jogo. Este projeto usa essa mesma linguagem:
* **HP (Health Points):** Representa o orçamento mensal. Gastar dinheiro é "tomar dano". Isso cria uma aversão à perda muito mais palpável do que ver um número vermelho no extrato bancário.
* **MP (Mana Points):** Representa as economias. Magia e poder em jogos exigem mana; na vida real, oportunidades e segurança exigem reservas financeiras.

### 2. Hackeando o Sistema de Recompensa (Dopamina)
O cérebro jovem é biologicamente focado em gratificação instantânea. Economizar dinheiro, por outro lado, é um exercício de paciência e recompensa a longo prazo. 
* O dashboard gamificado preenche essa lacuna através de **Missions (Missões)** e **Quests (Tarefas)**. 
* Ao não gastar, o usuário não está apenas "guardando dinheiro" no vácuo; ele está ativamente ganhando XP, completando missões como "Sobreviva ao Mês" e recebendo feedback visual imediato e positivo (cores neon brilhantes, ícones de sucesso).

### 3. Educação Financeira Disfarçada de "Skill Tree" (Árvore de Habilidades)
Um dos maiores bloqueios para jovens adultos é entender os juros compostos e os investimentos. O projeto aborda isso na aba de **Upgrades**:
* Ao invés de palestras complexas sobre "Fundo de Emergência" ou "Tesouro Direto", o aplicativo trata essas modalidades como *habilidades desbloqueáveis*. 
* O usuário entende que precisa acumular um recurso (dinheiro) para "comprar" um upgrade (investimento) que lhe dará um buff passivo (rendimento ou segurança). Isso naturaliza o conceito de investir desde cedo.

### 4. Redução da Ansiedade Financeira
Lidar com dinheiro gera ansiedade. Ao envelopar a gestão financeira em uma estética Cyberpunk/Gamificada, o projeto reduz a intimidação. O usuário deixa de ser um "devedor" olhando para contas a pagar e passa a ser um **Jogador (Player)** administrando seu inventário. Erros financeiros deixam de ser fracassos pessoais e se tornam apenas obstáculos de uma fase superável.

## Tecnologias Utilizadas

* **React (com Vite):** Biblioteca principal para renderização de interfaces rápidas e modulares.
* **TypeScript:** Adiciona tipagem estática, garantindo um código mais seguro e previsível.
* **Tailwind CSS (v3):** Framework utilitário usado para construir toda a interface responsiva e o design system neon/cyberpunk.
* **Lucide React:** Biblioteca de ícones vetoriais leves e customizáveis.
